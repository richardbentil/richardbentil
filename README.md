Hi 👋 I'm Richard Bentil

Senior Software Engineer · Accra, Ghana

I build full-stack production systems where being wrong has real consequences — payments, payroll, field data collection. I work frontend-first but follow problems all the way down to the data layer.

- 🌍  Based in Accra, Ghana
- 🖥️  Portfolio at [richardbentil.com](http://richardbentil.com)

- 🤝  Open to senior full-stack / systems engineering roles and contracts
- ⚡  I design before I code — boundaries, data ownership, and failure modes first

### What I build

- **Payment systems** — card, mobile money, and QR rails (Digopay, MyGhQr)
- **Payroll & HR platforms** — auditable payroll runs, employee records, PDF payslips
- **Field data collection** — offline-capable mobile apps syncing to a central backend
- **Auth & multi-tenancy** — JWT, RBAC, tenant isolation at the data layer
